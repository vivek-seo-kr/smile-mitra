# Smile Mitra Dental & Aesthetic Clinic

A clean, fast, and professional single-page portfolio website for **Smile Mitra Dental & Aesthetic Clinic**, providing top-quality, affordable dental and aesthetic care in Kangra, Himachal Pradesh.

🔗 **Live Demo:** [your-link-here.github.io](https://your-link-here.github.io)

---

## About

This portfolio is designed to help patients find and connect with Smile Mitra Dental & Aesthetic Clinic online. It highlights the clinic's dental and aesthetic services, making it easy for patients to book appointments and get directions to the clinic.

## Sections

- **Hero** — Introduction with clinic name and call-to-action
- **Services** — Dental and aesthetic treatments offered
- **About** — Clinic philosophy and patient care approach
- **Testimonials** — Patient reviews and feedback
- **Contact** — Address, phone, and Google Maps location
- **Footer** — Social media links and clinic info

## Services Covered

- 🦷 General Dentistry
- ✨ Aesthetic Treatments
- 🪥 Dental Hygiene & Cleaning
- 😁 Smile Makeover

## Tech Stack

- Pure HTML, CSS & vanilla JavaScript
- [Tailwind CSS](https://tailwindcss.com/) via CDN
- [Lucide Icons](https://unpkg.com/lucide@latest) for UI icons
- [Google Fonts](https://fonts.google.com/) for typography
- No frameworks · No build tools · No dependencies

## Deployment

This project is deployed via **GitHub Pages**.

To deploy your own copy:

1. Fork or clone this repository
2. Go to **Settings → Pages**
3. Set source to `main` branch, `/ (root)`
4. Your site will be live at `https://your-username.github.io/repo-name`

## Local Preview

Just open `index.html` in any browser — no setup required.

```bash

cd your-repo-name

```
